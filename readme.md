# Webflow Landing Page

A modern, responsive landing page built with Webflow and exported as a static website. The project is designed to provide a clean user experience, fast loading times, and responsive layouts across desktop, tablet, and mobile devices.

## Features

* Responsive design
* Modern and clean UI
* Optimized layout for conversions
* Cross-browser compatibility
* Fast-loading static website
* Easy deployment to hosting platforms

## Technologies Used

* HTML5
* CSS3
* JavaScript
* Webflow

## Project Structure

```text
webflow-landing-page/
├── css/
├── js/
├── images/
├── index.html
└── README.md
```

## Getting Started

1. Clone the repository:

```bash
git clone https://github.com/donaldodeh22-web/webflow-landing-page.git
```

2. Navigate to the project directory:

```bash
cd webflow-landing-page
```

3. Open `index.html` in your browser.

## Deployment

This project can be deployed on:

* GitHub Pages
* Netlify
* Vercel
* Any static web hosting service
